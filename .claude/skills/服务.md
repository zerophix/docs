# 后台服务管理

Learning Log 前后端由 macOS launchd 管理，开机自启，崩溃自动恢复。

## 命令

```bash
learnlog service status      # 查看服务运行状态
learnlog service restart     # 重启所有服务
learnlog service logs        # 查看最近日志
learnlog service install     # 安装/重新安装服务
learnlog service uninstall   # 卸载服务（恢复手动模式）
```

## 关键路径

- 后端 plist: `~/Library/LaunchAgents/com.learning-log.backend.plist`
- 前端 plist: `~/Library/LaunchAgents/com.learning-log.frontend.plist`
- 日志目录: `~/.learning-log/`

## 验证

```bash
launchctl list | grep learning-log   # 确认已加载
curl -s localhost:8002/api/stats     # 后端健康检查
curl -s -o /dev/null -w '%{http_code}' localhost:3000  # 前端健康检查
```
