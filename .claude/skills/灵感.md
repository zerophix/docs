# 顿悟捕获器

快速捕获当前讨论中的顿悟时刻。**内容必须完整展开，不能只写一句话。**

## 流程

### 1. 在对话中完整展开灵感（≥500字）

必须包含以下部分，全部合并为一篇完整文本：

**核心洞察**（2-3段）
- 这个顿悟的本质是什么？
- 为什么重要？它解决了什么问题？之前为什么没意识到？

**类比说明**（1段）
- 用日常概念类比这个洞察，让外行也能理解

**应用场景**（2-3个）
- 能在哪些具体场景中使用？每个场景一句话说明

**可迁移模式**（1段）
- 抽象出通用方法论，能复用到其他领域吗？

太短（<200字）则追问展开，不要急着入库。

### 2. 入库

通过 MCP 工具 `quick_capture` 保存：
- `topic`: 灵感主题
- `insight`: 上面展开的完整内容（核心洞察+类比+场景+迁移模式，全部合并）

**如果 MCP 不可用**，使用管道方式（避免长文截断）:
```bash
echo "完整灵感内容..." | learnlog record "主题" --pipe --energy 5 --aha true --source "aha-capture"
```

完成后告知 ID + http://localhost:3000
